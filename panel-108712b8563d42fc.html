Domain Lookup Info

How domain checks work:

  • With Porkbun API keys: Real-time availability + exact promo pricing, premium domain detection, and renewal rates directly from Porkbun.
  • Without keys: Falls back to tldx for basic availability + cached Porkbun bulk pricing.

Note: Porkbun rate-limits domain checks to ~1 per 10 seconds. The system checks all TLDs in parallel — some get Porkbun data, the rest use tldx as fallback.
